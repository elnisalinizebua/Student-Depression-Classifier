This project focuses on classifying depression levels among students using Machine Learning techniques.  
The study compares two algorithms: **C4.5 (Decision Tree)** and **Random Forest**, developed as part of an academic research project.
(Project ini dibuat untuk mengklasifikasikan tingkat depresi pada pelajar menggunakan algoritma Machine Learning. Penelitian ini membandingkan dua metode yaitu C4.5 (Decision Tree) dan Random Forest, dan merupakan bagian dari penelitian skripsi saya. Repository ini berisi notebook pelatihan model, aplikasi Streamlit sederhana untuk melakukan prediksi data baru, serta contoh dataset dan dokumentasi cara menjalankan project).

The repository includes:
- Machine Learning model training 
- A simple **Streamlit app** for predicting new data
- Sample dataset
- Documentation and instructions for running the project
